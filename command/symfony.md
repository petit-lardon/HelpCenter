# Symfony

[&#8672; *Table of contents*](/helpCenter/)

## Launch web server

```bash
php bin/console server:run
```

## Launch application and watch for changes

```bash
npm run watch
```

## Clear cache

```bash
php bin/console cache:clear
```

## Update translations

```bash
php bin/console trans:update --force --clean fr BridgeFrontBundle
```
